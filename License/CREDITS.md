# Благодарности

## Mastra Framework

Сердечная благодарность команде **Mastra** ([mastra.ai](https://mastra.ai)) за создание мощного и гибкого TypeScript-фреймворка для построения AI-агентов. Mastra обеспечивает надёжную основу для агентов Ufi и Oscar, их инструментов и оркестрации.

## Агент Ufi

Основной агент Ufi работает на **Qwen3-32B** через Groq API. Спасибо:
- **Qwen Team (Alibaba)** — за открытую и мощную языковую модель
- **Groq** — за молниеносный инференс

## Агент Oscar (Deep Research)

Агент Oscar для глубинного поиска работает на модели **opencode/big-pickle**. Спасибо:
- **OpenCode** — за платформу и API
- Разработчикам инструментов **webSearch** и **browserRead**, которые дают Oscar возможность исследовать интернет

## Экосистема

- **Google Gemini** — за мультимодальный AI и Bidirectional WebSocket API
- **Telegram (GramJS)** — за библиотеку MTProto для работы с Telegram
